# RouteSignal Pitch Deck

This folder contains the hackathon pitch artifact for RouteSignal.

- `routesignal-hackathon-pitch.pptx` is the editable PowerPoint deck.
- `routesignal-hackathon-pitch.md` is the source outline and talk track.

The deck uses local RouteSignal data from `site/public/data/routesdb.json`, generated on 2026-05-21 from public x402 route records plus RouteSignal derived fields.
